<h1>How-To</h1>

1. clone repo `git clone git@github.com:H34Dlezman/node-ytdl-core-playlists.git`
2. `cd node-ytdl-core-playlists `
3. `npm install`
4. `node index.js`

<p>You'll find the videos in "./vids/"</p>
